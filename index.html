Welcome to Waheed's OpenGL Project

This project outlines a set of OpenGL code samples, written in C/C++. Ideally this toolbox will be used as a resource for graphics and game engine development. The tools are listed as follows:

  • OpenGL Dev Environment Setup (Ubuntu Linux)
  • Shaders
  • Points, Lines and Vertices
  • Colors, Pixels and Framebuffers
  • Cameras and Transformations
  • Textures
  • Lighting
